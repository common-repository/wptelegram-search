== Description ==
Let people search on your wordpress website using your bot, both inline as well as text search.

You can increase the audience of your website by providing people the facility to search for its content right in Telegram using your bot. So, the bot will do your job.

**IMPORTANT**
This plugin requires SSL to set a webhook for receiving updates

**Some Features**

*	Allows inline search (don't forget to send `/setinline` command to @BotFather)
*	Allows text message search
*	You can use `/search` command followed by the text to search in groups
*	You can control what to search and what not to
*	Filter posts by authors, categories or custom  taxonomy terms
*	Message Template composer with Emojis
*	Supports search in Custom Post Types

By default setup, your bot will respond to three commands and their aliases
**Default Commands**

*	`/start` alias: `/hi`, `/hello`
*	`/help` alias: `/listcommands`
*	`/search` alias: `/s`, `/find`

== Installation ==


1. Upload the `wptelegram-search` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the Plugins menu in WordPress. After activation, you should see the menu of this plugin the the admin
3. Configure the plugin.

**Enjoy!**
